MestreGrana: IA Resiliente para Mentoria Financeira (Desafio Santander DIO)

## Texto

🚀 Projeto final do bootcamp Santander DIO publicado!

Apresento o MestreGrana, uma IA resiliente para mentoria financeira, criada para escalar em ambientes corporativos, bootcamps e consultorias. O sistema une failover multi-LLM (Groq, Gemini, OpenAI), RAG contextual, UX acessível e engenharia de prompt avançada para entregar recomendações seguras, personalizadas e sem alucinação.

🔹 Resiliência: failover automático entre modelos, garantindo disponibilidade
🔹 RAG: respostas fundamentadas nos dados reais do cliente
🔹 UX: chat interativo, áudio em português, acessibilidade mobile
🔹 Engenharia de Prompt: few-shot, edge cases, neutralidade e transparência
🔹 Documentação completa, métricas e diagrama Mermaid
🔹 Visão de futuro: integração com grafos (Neo4j) para análise de vínculos e recomendações inteligentes
